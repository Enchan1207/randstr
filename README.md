# randstr

## Overview
the most simplest random string generator.

## Development Environment
 * Langguage:C
 * Execute:Windows 10 / sakura server(FreeBSD)
 
## Contents
 * main.c: main source
 
## Usage
 1. compile and generate execution file
 2. run `(generated file name) 15` to generate 15-letter string.
